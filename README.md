# Fakebook Backend (express & mongodb)

#### (This is a personal project to learn/explore MERN stack)

## Getting started:

- Ensure Node is installed
- Clone the repo
- In fakebook folder create a config folder with dev.env file in it
- Set up environemnt variables: PORT:value, MONGODB_URL=value, JWT_SECRET=value
- Install dependencies: npm install
- Setup a cluster in MongoDb Atlas and connect to express server
- To start local deveopment server:
  - npm run dev
